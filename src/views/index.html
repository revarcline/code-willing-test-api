Pig Latinizer

This is an API to translate english-language phrases to Pig Latin.

There is one Route.

POST /api/phrases/piglatin: accepts a json object with the key phrase with a value consisting only of alphabetic characters and spaces.

Sample request body:

{ "phrase": "The rain in Spain falls mainly on the plain" }

Sample response body:

"fullPhrase": {
"original": "The rain in Spain falls mainly on the plain",
"pigLatin": "Hetay ainray inway Painsay allsfay ainlymay onway hetay lainpay"
}
